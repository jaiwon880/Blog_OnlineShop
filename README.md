### 📊 Background

온라인 샵에서는 쇼핑, 커뮤니티 기능이 없고, 공지사항 게시판, 로그인한 사용자끼리 소통할 수 있는 커뮤니티(후기, 공지 게시판)가 없다.   

‘나’를 소개하는 공간 (포토 프레임) 등을 만들면 쇼핑몰, 소통, 일상 공유를 원하는 사용자들에게 즐거움을 전할 수 있을 것이다 라는 생각에서 아이디어를 기획함
